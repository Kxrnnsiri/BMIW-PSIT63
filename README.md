# BMIW
---
## What is BMIW ?
 BMIW( BMI & Workout ) คือ เป็นโปรเจคในวิชา PSIT โดยตัว BMIW โปรแกรมชนิดหนึ่งที่คำนวณ BMI โดยใช้ข้อมูลที่เกี่ยวกับน้ำหนักเเละส่วนสูงของผู้ใช้งานรวมถึง การออกกำลังกายเพื่อสร้างเสริมสุขภาพให้ดีขึ้นตามลำดับเเละสามารถ ระบุส่วนต่างๆที่ต้องการออกกำลังกายมากเป็นพิเศษ โดยออกเเบบมาให้เข้าใจง่าย เเละใช้ได้กับทุกคนที่มีความสนใจในด้านสุขภาพ
 ## Objective of this project
  - การได้ลองเขียนโปรแกรมpython เเละลองเรียนรู้จากใน lecture
  - ทดลองการทำงานเป็นรูปแบบ เป็นระบบ เพื่อพัฒนาตัวเองในอนาคตเเละทำให้เราสามารถทำงานร่วมกับคนอื่นได้
  - เกิดเเก้ไขเฉพาะหน้าได้เเละสามารถจัดการกับปัญหานั้นได้
  - เกิดการประยุกต์ต่อยอดให้ดีขึ้นไปในอนาคต
